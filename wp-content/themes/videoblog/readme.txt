== Description ==

Videoblog is a traditional blog/magazine theme that displays post in a 4-columns grid. Great for websites with a lot of posts.
The Front Page provides the ability to highlight your main posts as well as feature up to 5 different post categories.


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= How to setup the front page like the demo site? =

The demo site URL: http://demo.ilovewp.com/videoblog/

When you first activate Videoblog, your homepage will display posts in a traditional blog format. If youâ€™d like to add the additional featured sections from the demo, you can do that by going to Appearance > Customize > Theme Options.

= Footer Area =

The footer area contains 4 widgetized columns of equal width.

== Quick Specs (all measurements in pixels) ==

	1. The main column width is 760.
	2. In full-width layout the content width is 1100.
	3. A widget in the sidebar is 300.
	4. The featured image on the front page and on pages works best with images at least 240 wide.
	5. Featured Images for posts should be at least 240 wide.
	6. The footer widget columns are of equal width: 245.
